# LangChain & LangGraph Learning Repo

Este repositorio contiene ejemplos prácticos y notas de aprendizaje sobre:
- LangChain
- LangGraph

## Objetivo
Aprender a construir aplicaciones LLM:
- Chains
- Agents
- Graph-based workflows
- Manejo de estado y memoria

## Tecnologías
- Python
- LangChain
- LangGraph
- OpenAI / Anthropic (via API)

## Setup
1. Clonar el repo
2. Crear entorno virtual
3. Copiar `.env.example` a `.env`
4. Instalar dependencias

⚠️ Las credenciales **no** se incluyen en el repositorio.
